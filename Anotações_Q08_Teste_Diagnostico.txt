Observação:
Número inteiro é aquele que não possui partes fracionárias/decimais.
Método para testar se o número é inteiro:
Calcular o resto (%) dividindo o número por 1. 
	Se(resto==0): Número inteiro
	Se(resto!=0) ou Senão: Número não é inteiro
Em código:
	if (num%1 == 0){
		cout << "Número inteiro\n";
	}	
	else{
		cout << "Número não inteiro\n";
	}
